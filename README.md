# Mist & Ember Exports

A sustainable, elegant, and immersive digital experience built with **Next.js 14**, **TypeScript**, and **TailwindCSS** — celebrating the essence of Sri Lanka’s natural heritage and craftsmanship.

---

## ✨ Overview

**Mist & Ember Exports** is a modern web project showcasing the authenticity, culture, and organic beauty of Sri Lanka.  
It combines minimal design, strong typography, and seamless animations to create a luxurious and nature-inspired digital experience.

Built with:
- ⚡️ **Next.js 14** (App Router)
- 💚 **TypeScript**
- 🎨 **TailwindCSS**
- 🧩 **Zustand** (state management)
- 🖼️ **Sharp** (image optimization)
- 🧠 Optional **CMS integration** (Sanity, Hygraph, or Contentful)
